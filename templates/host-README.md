# Projeto Marketing Doma

Tudo **local nesta pasta** — sem instalação global.

## Setup (1×)

```bash
npm init -y
npm install marketing-doma-cli
npm run doma:install
```

## Criar post

```
/marketing-doma
```

Cursor: *"cria post Doma"*

## Comandos

```bash
npm run doma:status
npm run doma:update
npm run doma:setup      # reparar
npm run doma:export
```
