# Negrito global = 600 (MEDIDO — era 800, grosso demais)

**Data:** 2026-07-16
**Descoberto em:** POST prejuizo-industria (correção do Patrick: "afinar o negrito — regra em todas as criações")
**Sintoma:** `TextoRico` renderizava `**bold**` com fontWeight 800 — muito mais grosso que o modelo.
**Medição (stroke width das hastes, POST 270 @ fs78):** regular = 5px · bold do modelo = **9px**. Nosso 800 → 12px+; 700 → 11px; **600 → 8-9px ✓**.
**Regra:** default do `boldWeight` no `TextoRico` (base de TODOS os componentes) = **600**. Pesos maiores só com medição do modelo específico (passar `boldWeight` explícito).
**Como validar:** medir stroke no render (runs de pixels escuros numa linha do glifo) e comparar com o modelo — não confiar no número do fontWeight.
**Aplicar em:** todas as categorias, todas as criações novas. Recriações 1:1 antigas: re-validar com compare.py se re-renderizadas.
